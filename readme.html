Project setup

### Install dependencies ###
Installs Bootstrap Sass, Font Awesome, jQuery, Dart Sass and the dev server from package.json. > npm install
### Development ###
Watches Sass and serves the site with live reload at http://localhost:8000 > npm run dev
### Build ###
Compiles the minified stylesheet for deployment. > npm run build
