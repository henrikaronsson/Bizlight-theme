Prerequisites

  • Node.js (version 20 or newer)

Project setup

Install dependencies

Installs Bootstrap 5, Bootstrap Icons, Vite and Dart Sass from package.json.

> npm install

Development

Starts the Vite dev server with hot reload at http://localhost:5173

> npm run dev

Build

Builds the optimized production site into dist/.

> npm run build

Preview

Serves the production build locally.

> npm run preview
